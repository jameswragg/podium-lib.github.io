Getting Started

Podium consist of two parts; Podlets and Layouts.

Podlets

If one are to write page fragments, or Podlets in Podium speak, one should be using the @podium/podlet module.

Layouts

If one are to write a layout to compose page fragments together, one should be using the @podium/layout module.
