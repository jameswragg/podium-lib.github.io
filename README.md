# Podium website

https://pages.github.schibsted.io/Podium/dist/en/

## Setup

-   `npm install`
-   `npm run dev` in one tab
-   `npm start` in another tab
-   visit `http://localhost:5000` in your browser

## Making changes

-   update files in `/src`
-   refresh browser window to see changes

## Publishing changes

-   run `npm run build`
-   commit changes
-   push
